Summary: Latitude vs. X

The purpose of this project was to plot the changes in weather as you get closer to the equator. In order perform the analysis, data was obtained from the OpenWeatherMap API for over 500 randomly generated city coordinates from across the world.

Matplotlib was used to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, humidity, cloudiness, and wind speed. This site summarizes the source data and visualizations from the analysis. For the original analysis refer to the following repository.
